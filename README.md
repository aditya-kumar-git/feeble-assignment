# Feeble Assignment

**Live URL:** [https://feeble-assignment-six.vercel.app/](https://feeble-assignment-six.vercel.app/)

A simple landing page built with Next.js.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript 5
- Tailwind CSS 4
- Motion (animation library)
- Fonts: General Sans (local)

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the dev server:
   ```
   npm run dev
   ```
3. Open `http://localhost:3000`
